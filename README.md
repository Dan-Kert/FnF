Адаптированная версия Friday Night Funkin' для мобильных устройств.

---

### Основные изменения
* **Реклама**: Полностью отключена (вырезана из кода).
* **Идентификация**: Пакет приложения изменен на `md.dankert.fnf`.
* **CI/CD**: Настроена автоматическая сборка через GitHub Actions для получения актуальных артефактов.
